# Welcome to Affirmations

 [Live Demo](https://affirm-vr.web.app/).

## Introduction

This application is designed for users that want to create a wall of positive affirmations. 

## How to Use
1. Go to https://affirm-vr.web.app/
2. Click the yellow 'Login' button to start login
3. Click the yellow 'Create' to create a affirmation  

## Technologies Used
- Frontend: React
- Storage: Firebase Storage 
- Database: Firestore 
- API: Express/Node.js
- Other: Firebase authentication, hosting, and functions
